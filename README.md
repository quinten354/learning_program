# Learning program

## What is this about?

 -  A program to learn words, sentences, grammar and more
 -  You can create a list of items, it must be 2 words (given and unknown)
 -  You can learn, review, and much more

## Install

1. execute installer
	Linux: give execute permissions: `chmod 700 setup_linux.py`
		   execute file: `./setup_linux.py`
				or
		   `python3 setup_linux.py`
	Windows: execute `setup_windows.cmd`
				or
		     `python3 setup_windows.py`

2. follow steps to install the program

3. run the program
    execute `learning_program` or how you have named this file
